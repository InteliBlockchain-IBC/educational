## 🎯 Objetivo

Este repositório centraliza materiais de estudo sobre fundamentos e prática em blockchain, com foco em uma aprendizagem progressiva que abrange:

- **Conceitos Fundamentais**: O que é blockchain, como funciona e por que é importante.
- **Vocabulário Técnico**: Termos essenciais para navegar no ecossistema.
- **Leitura Guiada**: Análise de artigos, whitepapers e documentações.
- **Aplicação Prática**: Exercícios e projetos para consolidar o conhecimento.

## 🚀 Como Começar a Aprender

Sugerimos a seguinte trilha para um aprendizado mais eficaz:

1.  **Alinhe o vocabulário**: Comece pelo [**Dicionário**](./dicionario/README.md) para se familiarizar com os termos.
2.  **Estude a base**: Siga para a primeira aula em [**Aulas/Tokenização**](./aulas/tokenizacao/README.md).
3.  **Aprofunde-se**: Explore os links e materiais em [**Material Adicional**](./material_adicional/README.md).

## 🤝 Como Contribuir

Sua contribuição é muito bem-vinda! Veja como você pode ajudar:

<details>
  <summary><strong>Sugerir melhorias ou relatar erros</strong></summary>
  
  Se você encontrou um erro, tem uma sugestão de melhoria ou um novo material, por favor, [**abra uma Issue**](https://github.com/InteliBlockchain-IBC/educational/issues/new).
  
  Descreva sua sugestão com o máximo de detalhes possível, incluindo links se necessário.
</details>

<details>
  <summary><strong>Contribuir com código ou conteúdo</strong></summary>
  
  Para adicionar novos materiais, corrigir textos ou melhorar o código, siga estes passos:
  
  1. **Faça o Fork** do repositório.
  2. **Crie uma Branch** para sua modificação (`git checkout -b feature/sua-feature`).
  3. **Faça o Commit** das suas mudanças (`git commit -m 'Adiciona novo material sobre X'`).
  4. **Envie para a sua Branch** (`git push origin feature/sua-feature`).
  5. **Abra um Pull Request** com uma descrição clara do que foi alterado.
</details>
