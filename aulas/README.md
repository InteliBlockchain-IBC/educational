## 🎯 Objetivo da Pasta

O objetivo desta seção é fornecer um local centralizado para todos os materiais didáticos utilizados nas aulas. Cada subpasta corresponde a uma aula específica e contém:

- **Material Principal**: O guia ou apresentação principal da aula.
- **Dicionário Associado**: Um glossário com termos técnicos relevantes para o tema da aula.
- **Códigos e Exemplos**: Snippets e projetos práticos discutidos.

## 🤝 Como Contribuir

Sua contribuição é muito bem-vinda! Veja como você pode ajudar:

<details>
  <summary><strong>Sugerir melhorias ou relatar erros</strong></summary>
  
  Se você encontrou um erro, tem uma sugestão de melhoria ou um novo material para uma aula, por favor, [**abra uma Issue**](https://github.com/seu-usuario/seu-repositorio/issues/new).
  
  Descreva sua sugestão com o máximo de detalhes possível, incluindo links se necessário.
</details>

<details>
  <summary><strong>Contribuir com conteúdo</strong></summary>
  
  Para adicionar novos materiais, corrigir textos ou melhorar os exemplos, siga estes passos:
  
  1. **Faça o Fork** do repositório.
  2. **Crie uma Branch** para sua modificação (`git checkout -b feature/sua-feature`).
  3. **Faça o Commit** das suas mudanças (`git commit -m 'Adiciona material sobre X na aula Y'`).
  4. **Envie para a sua Branch** (`git push origin feature/sua-feature`).
  5. **Abra um Pull Request** com uma descrição clara do que foi alterado.
</details>
