# Glossário de Termos - Blockchain e Criptografia

## Termos essenciais

| Termo              | Definição                                                                                                                     |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------- |
| **Blockchain**     | Ledger distribuído e imutável onde transações são agrupadas em blocos criptograficamente ligados.                             |
| **Smart Contract** | Programa executável em uma blockchain que automatiza a execução de acordos sem intermediários.                                |
| **EVM**            | Ethereum Virtual Machine; máquina virtual que executa bytecode de contratos inteligentes em Ethereum e redes EVM-compatíveis. |
| **Gas**            | Unidade de medida de custo computacional em Ethereum; quanto maior a complexidade, maior o gas necessário.                    |
| **Consenso**       | Mecanismo que garante que todos os nós da rede concordem sobre o estado válido do ledger (ex: Proof of Work, Proof of Stake). |
| **DApp**           | Aplicação descentralizada que roda em uma blockchain e combina smart contracts com interface de usuário.                      |
| **Testnet**        | Rede de teste paralela à rede principal, usada para experimentar sem riscos de perda real de recursos.                        |
| **Wallet**         | Programa ou dispositivo que gerencia chaves criptográficas e permite enviar/receber criptomoedas.                             |
| **Mempool**        | Pool de memória onde transações pendentes aguardam confirmação na blockchain.                                                 |
| **Token**          | Representação digital de valor emitida em uma blockchain, podendo ser moeda, ativo ou direito.                                |

## Termos intermediários

| Termo                    | Definição                                                                                                                     |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------- |
| **DeFi**                 | Finanças Descentralizadas; ecosistema de aplicações financeiras rodando em blockchains sem intermediários tradicionais.       |
| **NFT**                  | Non-Fungible Token; token único e não intercambiável que representa propriedade de ativos digitais.                           |
| **Whitepaper**           | Documento técnico que apresenta a visão, arquitetura e mecanismos de uma rede ou projeto blockchain.                          |
| **Fork**                 | Modificação do protocolo de uma blockchain; pode ser compatível com versões anteriores (soft fork) ou não (hard fork).        |
| **Nó**                   | Computador que executa o software da blockchain e participa da rede mantendo uma cópia do ledger.                             |
| **Merkle Tree**          | Estrutura de dados que agrupa e resume informações de forma que mudanças em um bloco sejam detectadas.                        |
| **Proof of Work (PoW)**  | Mecanismo de consenso que requer resolução de problemas criptográficos para validar blocos (usado em Bitcoin).                |
| **Proof of Stake (PoS)** | Mecanismo de consenso onde validadores são escolhidos com base em quantas moedas possuem e "apostam" (usado em Ethereum 2.0). |

## Termos avançados

| Termo                    | Definição                                                                                                                     |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------- |
| **Sidechain**            | Blockchain paralela à rede principal, permitindo transações mais rápidas com menor custo.                                     |
| **Rollup**               | Solução de escalabilidade que agrupa múltiplas transações fora da chain e as registra na chain principal de forma compactada. |
| **MEV**                  | Miner/Maximal Extractable Value; valor que pode ser extraído reorganizando transações em um bloco.                            |
| **Oráculos**             | Serviços que trazem dados do mundo real para dentro da blockchain de forma segura e descentralizada.                          |
| **Bridges**              | Contratos ou protocolos que permitem transferência de ativos entre diferentes blockchains.                                    |
| **Sharding**             | Técnica de escalabilidade que divide a blockchain em múltiplas cadeias paralelas (shards).                                    |
| **Zero-Knowledge Proof** | Prova criptográfica que valida informações sem revelar os dados subjacentes.                                                  |
| **Staking**              | Processo de bloquear criptomoedas em um protocolo para validar transações e receber recompensas.                              |
| **Slashing**             | Penalidade aplicada a validadores que violam regras do protocolo, resultando em perda de seus fundos.                         |
| **Cross-Chain**          | Protocolos que permitem interação e comunicação entre diferentes blockchains.                                                 |

## Acrônimos comuns

| Acrônimo | Significado                                                                          |
| -------- | ------------------------------------------------------------------------------------ |
| **BIP**  | Bitcoin Improvement Proposal; proposta de melhoria para o protocolo Bitcoin.         |
| **EIP**  | Ethereum Improvement Proposal; proposta de melhoria para o protocolo Ethereum.       |
| **SIP**  | Stellar Improvement Proposal; proposta de melhoria para o protocolo Stellar.         |
| **DAO**  | Decentralized Autonomous Organization; organização automatizada por smart contracts. |
| **DEX**  | Decentralized Exchange; bolsa descentralizada para troca de tokens.                  |
| **CEX**  | Centralized Exchange; bolsa centralizada controlada por uma empresa ou entidade.     |
| **KYC**  | Know Your Customer; processo de verificação de identidade.                           |
| **AML**  | Anti-Money Laundering; conformidade contra lavagem de dinheiro.                      |
| **API**  | Application Programming Interface; interface para comunicação entre programas.       |
| **CLI**  | Command Line Interface; interface de linha de comando.                               |
| **SDK**  | Software Development Kit; conjunto de ferramentas para desenvolvimento.              |
